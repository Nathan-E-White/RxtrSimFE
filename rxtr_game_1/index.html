Reactor Simulation

This is a simulation of a nuclear reactor. The reactor is a 2D grid of cells, each of which can be either empty, contain a neutron, or contain a fuel pellet. The simulation is run by repeatedly applying the following rules to the grid:

  1. Neutrons move in a random direction.
  2. Neutrons are absorbed by fuel pellets.
  3. Neutrons are absorbed by empty cells.
  4. Neutrons are reflected by the edges of the grid.
  5. When a neutron is absorbed by a fuel pellet, the pellet is destroyed and a new neutron is created.

The simulation is run by repeatedly applying the above rules to the grid. The grid is displayed as a table, with each cell represented by a table cell. The table is updated by changing the class of each cell to reflect the state of the corresponding cell in the grid.
